## 四大功能支柱

UpWord 的文章阅读器围绕 **四个核心功能支柱** 组织：

<CardGrid stagger>
	<Card title="Reading 阅读" icon="open-book">
		难度切换（简单/中等/困难）和阅读进度追踪。
	</Card>
	<Card title="Syntax 语法" icon="pencil">
		语法分析，可视化注解与语法高亮。
	</Card>
	<Card title="Echoes 回响" icon="approve-check-circle">
		悬停词汇时显示跨文章历史语境。
	</Card>
	<Card title="Audio 音频" icon="rss">
		TTS 预加载、音频播放器与 Edge TTS 客户端支持。
	</Card>
</CardGrid>

## 技术栈

<CardGrid>
	<Card title="前端" icon="laptop">
		**Astro** + React 组件，部署于 Cloudflare Pages。
	</Card>
	<Card title="后端" icon="seti:db">
		**Elysia** (Bun) + SQLite/D1/Turso，通过 Docker 部署。
	</Card>
	<Card title="LLM 管线" icon="star">
		多阶段文章生成，支持难度自适应。
	</Card>
	<Card title="任务队列" icon="list-format">
		后台 Worker 异步处理 LLM 任务。
	</Card>
</CardGrid>
